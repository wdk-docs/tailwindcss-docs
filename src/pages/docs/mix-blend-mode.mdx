## 基本用法

### 设置元素的混合模式

使用`mix-blend-{mode}`实用工具来控制元素的内容如何与背景混合。

```html {{ example: true }}
<div class="flex justify-center -space-x-14">
  <div class="w-32 h-32 mix-blend-multiply bg-blue-400 rounded-full"></div>
  <div class="w-32 h-32 mix-blend-multiply bg-pink-400 rounded-full"></div>
</div>
```

```html
<div class="flex justify-center -space-x-14">
  <div class="**mix-blend-multiply** bg-blue-400 ..."></div>
  <div class="**mix-blend-multiply** bg-pink-400 ..."></div>
</div>
```
