## 基本用法

### 优化将会改变

使用`will-change-scroll`、`will-change-contents`和`will-change-transform`来优化在不久的将来可能发生变化的元素，指示浏览器在实际开始之前准备必要的动画。

```html
<div class="overflow-auto **will-change-scroll**">
  <!-- ... -->
</div>
```

建议您在元素更改之前应用这些实用工具，然后在它完成后不久使用`will-change-auto`删除它。

will-change属性用于处理**已知性能问题**时的最后手段。避免过多地使用这些实用程序，或者只是为了预期性能问题而使用这些实用程序，因为这实际上可能导致页面的性能降低。

## 使用自定义值

### 自定义主题

默认情况下，Tailwind提供了四个`will-change`实用程序。你可以通过在你的`tailwind.config.js`文件中编辑`theme.willChange`或`theme.extend.willChange`来定制这些值。

```diff-js {{ filename: 'tailwind.config.js' }}
  module.exports = {
    theme: {
      extend: {
+       willChange: {
+         'left-top': 'left, top',
+       }
      }
    }
  }
```

在[theme customization](/docs/theme#customization-the-default-theme)文档中了解更多关于自定义默认主题的信息。
