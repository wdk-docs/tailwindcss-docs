## 基本用法

### Collapse

Use `border-collapse` to combine adjacent cell borders into a single border when possible. Note that this includes collapsing borders on the top-level `<table>` tag.

```html {{ example: { p: 'none' } }}
<div class="px-4 py-8 sm:px-8">
  <table class="border-collapse w-full border border-slate-400 dark:border-slate-500 bg-white dark:bg-slate-800 text-sm shadow-sm">
    <thead class="bg-slate-50 dark:bg-slate-700">
      <tr>
        <th class="w-1/2 border border-slate-300 dark:border-slate-600 font-semibold p-4 text-slate-900 dark:text-slate-200 text-left">State</th>
        <th class="w-1/2 border border-slate-300 dark:border-slate-600 font-semibold p-4 text-slate-900 dark:text-slate-200 text-left">City</th>
      </tr>
    </thead>
    <tbody>
      <tr>
        <td class="border border-slate-300 dark:border-slate-700 p-4 text-slate-500 dark:text-slate-400">Indiana</td>
        <td class="border border-slate-300 dark:border-slate-700 p-4 text-slate-500 dark:text-slate-400">Indianapolis</td>
      </tr>
      <tr>
        <td class="border border-slate-300 dark:border-slate-700 p-4 text-slate-500 dark:text-slate-400">Ohio</td>
        <td class="border border-slate-300 dark:border-slate-700 p-4 text-slate-500 dark:text-slate-400">Columbus</td>
      </tr>
      <tr>
        <td class="border border-slate-300 dark:border-slate-700 p-4 text-slate-500 dark:text-slate-400">Michigan</td>
        <td class="border border-slate-300 dark:border-slate-700 p-4 text-slate-500 dark:text-slate-400">Detroit</td>
      </tr>
    </tbody>
  </table>
</div>
```

```html
<table class="**border-collapse** border border-slate-400 ..." dark-class="**border-collapse** border border-slate-500 ...">
  <thead>
    <tr>
      <th class="border border-slate-300 ..." dark-class="border border-slate-600 ...">State</th>
      <th class="border border-slate-300 ..." dark-class="border border-slate-600 ...">City</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td class="border border-slate-300 ..." dark-class="border border-slate-700 ...">Indiana</td>
      <td class="border border-slate-300 ..." dark-class="border border-slate-700 ...">Indianapolis</td>
    </tr>
    <tr>
      <td class="border border-slate-300 ..." dark-class="border border-slate-700 ...">Ohio</td>
      <td class="border border-slate-300 ..." dark-class="border border-slate-700 ...">Columbus</td>
    </tr>
    <tr>
      <td class="border border-slate-300 ..." dark-class="border border-slate-700 ...">Michigan</td>
      <td class="border border-slate-300 ..." dark-class="border border-slate-700 ...">Detroit</td>
    </tr>
  </tbody>
</table>
```
