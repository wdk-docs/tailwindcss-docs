## 基本用法

### 禁用文本选择

Use `select-none` to prevent selecting text in an element and its children.

```html {{ example: { hint: 'Try selecting the text to see the expected behaviour' } }}
<div class="flex justify-center">
  <div class="select-none inline-flex text-center bg-white text-slate-900 font-semibold font-sans text-sm shadow-lg rounded-lg ring-1 ring-slate-900/5 px-4 py-3 dark:bg-slate-800 dark:ring-0 dark:highlight-white/5 dark:text-slate-200">
    The quick brown fox jumps over the lazy dog.
  </div>
</div>
```

```html
<div class="**select-none** ...">
  The quick brown fox jumps over the lazy dog.
</div>
```

### 允许文本选择

Use `select-text` to allow selecting text in an element and its children.

```html {{ example: { hint: 'Try selecting the text to see the expected behaviour' } }}
<div class="flex justify-center">
  <div class="select-text inline-flex text-center bg-white text-slate-900 font-semibold font-sans text-sm shadow-lg rounded-lg ring-1 ring-slate-900/5 px-4 py-3 dark:bg-slate-800 dark:ring-0 dark:highlight-white/5 dark:text-slate-200">
    The quick brown fox jumps over the lazy dog.
  </div>
</div>
```

```html
<div class="**select-text** ...">
  The quick brown fox jumps over the lazy dog.
</div>
```

### 在一次点击中选择所有文本

Use `select-all` to automatically select all the text in an element when a user clicks.

```html {{ example: { hint: 'Try selecting the text to see the expected behaviour' } }}
<div class="flex justify-center">
  <div class="select-all inline-flex text-center bg-white text-slate-900 font-semibold font-sans text-sm shadow-lg rounded-lg ring-1 ring-slate-900/5 px-4 py-3 dark:bg-slate-800 dark:ring-0 dark:highlight-white/5 dark:text-slate-200">
    The quick brown fox jumps over the lazy dog.
  </div>
</div>
```

```html
<div class="**select-all** ...">
  The quick brown fox jumps over the lazy dog.
</div>
```

### 使用自动选择行为

Use `select-auto` to use the default browser behavior for selecting text. Useful for undoing other classes like `select-none` at different breakpoints.

```html {{ example: { hint: 'Try selecting the text to see the expected behaviour' } }}
<div class="flex justify-center">
  <div class="select-auto inline-flex text-center bg-white text-slate-900 font-semibold font-sans text-sm shadow-lg rounded-lg ring-1 ring-slate-900/5 px-4 py-3 dark:bg-slate-800 dark:ring-0 dark:highlight-white/5 dark:text-slate-200">
    The quick brown fox jumps over the lazy dog.
  </div>
</div>
```

```html
<div class="**select-auto** ...">
  The quick brown fox jumps over the lazy dog.
</div>
```
