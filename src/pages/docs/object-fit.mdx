## 基本用法

### 调整大小以覆盖容器

Resize an element's content to cover its container using `object-cover`.

```html {{ example: true }}
<div class="text-center rounded-lg overflow-hidden w-56 sm:w-96 mx-auto">
  <img class="object-cover h-48 w-full " src="https://images.unsplash.com/photo-1554629947-334ff61d85dc?ixid=MnwxMjA3fDB8MHxwaG90by1wYWdlfHx8fGVufDB8fHx8&ixlib=rb-1.2.1&auto=format&fit=crop&w=1024&h=1280&q=80" />
</div>
```

```html
<div class="bg-indigo-300 ...">
  <img class="**object-cover** h-48 w-96 ...">
</div>
```

### 包含一个元素

Resize an element's content to stay contained within its container using `object-contain`.

```html {{ example: true }}
<div class="relative rounded-lg text-center overflow-hidden w-56 sm:w-96 mx-auto">
  <div class="absolute inset-0 opacity-50 bg-stripes-gray"></div>
  <img class="relative z-10 object-contain h-48 w-full" src="https://images.unsplash.com/photo-1554629947-334ff61d85dc?ixid=MnwxMjA3fDB8MHxwaG90by1wYWdlfHx8fGVufDB8fHx8&ixlib=rb-1.2.1&auto=format&fit=crop&w=1024&h=1280&q=80" />
</div>
```

```html
<div class="bg-purple-300 ...">
  <img class="**object-contain** h-48 w-96 ...">
</div>
```

### 拉伸以适应容器的

Stretch an element's content to fit its container using `object-fill`.

```html {{ example: true }}
<div class="bg-stripes-sky-blue rounded-lg text-center overflow-hidden w-56 sm:w-96 mx-auto">
  <img class="object-fill h-48 w-full" src="https://images.unsplash.com/photo-1554629947-334ff61d85dc?ixid=MnwxMjA3fDB8MHxwaG90by1wYWdlfHx8fGVufDB8fHx8&ixlib=rb-1.2.1&auto=format&fit=crop&w=1024&h=1280&q=80" />
</div>
```

```html
<div class="bg-sky-300 ...">
  <img class="**object-fill** h-48 w-96 ...">
</div>
```

### 如果太大，就按比例缩小

Display an element's content at its original size but scale it down to fit its container if necessary using `object-scale-down`.

```html {{ example: true }}
<div class="relative rounded-lg text-center overflow-hidden w-56 sm:w-96 mx-auto">
  <div class="absolute inset-0 opacity-50 bg-stripes-gray"></div>
  <img class="relative z-10 object-scale-down h-48 w-full" src="https://images.unsplash.com/photo-1554629947-334ff61d85dc?ixid=MnwxMjA3fDB8MHxwaG90by1wYWdlfHx8fGVufDB8fHx8&ixlib=rb-1.2.1&auto=format&fit=crop&w=128&h=160&q=80" />
</div>
```

```html
<div class="bg-cyan-300">
  <img class="**object-scale-down** h-48 w-96 ...">
</div>
```

### 使用元素的原始大小

Display an element's content at its original size ignoring the container size using `object-none`.

```html {{ example: true }}
<div class="relative rounded-lg text-center overflow-hidden w-56 sm:w-96 mx-auto">
  <div class="absolute inset-0 opacity-50 bg-stripes-gray"></div>
  <img class="relative z-10 object-none h-48 w-full" src="https://images.unsplash.com/photo-1554629947-334ff61d85dc?ixid=MnwxMjA3fDB8MHxwaG90by1wYWdlfHx8fGVufDB8fHx8&ixlib=rb-1.2.1&auto=format&fit=crop&w=1024&h=1280&q=80" />
</div>
```

```html
<div class="bg-yellow-300">
  <img class="**object-none** h-48 w-96 ...">
</div>
```
