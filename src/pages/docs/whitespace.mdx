## 基本用法

### 正常的

使用`whitespace-normal`使文本在元素内正常换行。换行符和空格将被折叠。

```html {{ example: { p: 'none' } }}
<div class="bg-white p-8 shadow-xl text-slate-700 max-w-md mx-auto overflow-auto text-sm leading-6 sm:text-base sm:leading-7 dark:bg-slate-800 dark:text-slate-400">
  <div class="whitespace-normal">Hey everyone!
It's almost 2022       and we still don't know if there       is aliens living among us, or do we? Maybe the person writing this is an alien.
You will never know.</div>
</div>
```

```html
<div class="w-3/4 ...">
  <div class="**whitespace-normal** ...">Hey everyone!

It's almost 2022       and we still don't know if there       is aliens living among us, or do we? Maybe the person writing this is an alien.

You will never know.</div>
</div>
```

### 没有包装

使用`whitespace-nowrap`来防止文本在元素内换行。换行符和空格将被折叠。

```html {{ example: { p: 'none' } }}
<div class="bg-white px-6 py-8 shadow-xl text-slate-700 max-w-md mx-auto overflow-auto text-sm leading-6 sm:text-base sm:leading-7 dark:bg-slate-800 dark:text-slate-400">
  <div class="whitespace-nowrap">Hey everyone!
It's almost 2022       and we still don't know if there       is aliens living among us, or do we? Maybe the person writing this is an alien.
You will never know.</div>
</div>
```

```html
<div class="w-3/4 overflow-x-auto ...">
  <div class="**whitespace-nowrap** ...">Hey everyone!

It's almost 2022       and we still don't know if there       is aliens living among us, or do we? Maybe the person writing this is an alien.

You will never know.</div>
</div>
```

### Pre

使用`whitespace-pre`在元素中保留换行符和空格。文本不会被换行。

```html {{ example: { p: 'none' } }}
<div class="bg-white px-6 py-8 shadow-xl text-slate-700 max-w-md mx-auto overflow-auto text-sm leading-6 sm:text-base sm:leading-7 dark:bg-slate-800 dark:text-slate-400">
  <div class="whitespace-pre">Hey everyone!

It's almost 2022       and we still don't know if there       is aliens living among us, or do we? Maybe the person writing this is an alien.

You will never know.</div>
</div>
```

```html
<div class="w-3/4 overflow-x-auto ...">
  <div class="**whitespace-pre** ...">Hey everyone!

It's almost 2022       and we still don't know if there       is aliens living among us, or do we? Maybe the person writing this is an alien.

You will never know.</div>
</div>
```

### Pre Line

使用`whitespace-pre-line`在元素中保留换行符，但不保留空格。文本将正常换行。

```html {{ example: { p: 'none' } }}
<div class="bg-white px-6 py-8 shadow-xl text-slate-700 max-w-md mx-auto overflow-auto text-sm leading-6 sm:text-base sm:leading-7 dark:bg-slate-800 dark:text-slate-400">
  <div class="whitespace-pre-line">Hey everyone!

It's almost 2022       and we still don't know if there       is aliens living among us, or do we? Maybe the person writing this is an alien.

You will never know.</div>
</div>
```

```html
<div class="w-3/4 ...">
  <div class="**whitespace-pre-line** ...">Hey everyone!

It's almost 2022       and we still don't know if there       is aliens living among us, or do we? Maybe the person writing this is an alien.

You will never know.</div>
</div>
```

### Pre Wrap

使用`whitespace-pre-wrap`在元素中保留换行符和空格。文本将正常换行。

```html {{ example: { p: 'none' } }}
<div class="bg-white px-6 py-8 shadow-xl text-slate-700 max-w-md mx-auto overflow-auto text-sm leading-6 sm:text-base sm:leading-7 dark:bg-slate-800 dark:text-slate-400">
  <div class="whitespace-pre-wrap">Hey everyone!

It's almost 2022       and we still don't know if there       is aliens living among us, or do we? Maybe the person writing this is an alien.

You will never know.</div>
</div>
```

```html
<div class="w-3/4 ...">
  <div class="**whitespace-pre-wrap** ...">Hey everyone!

It's almost 2022       and we still don't know if there       is aliens living among us, or do we? Maybe the person writing this is an alien.

You will never know.</div>
</div>
```

### 断点空白

使用`whitespace-break-spaces`在元素中保留换行符和空格。行尾的空白不会挂起，而是会自动换行到下一行。

```html {{ example: { p: 'none' } }}
<div class="bg-white px-6 py-8 shadow-xl text-slate-700 max-w-md mx-auto overflow-auto text-sm leading-6 sm:text-base sm:leading-7 dark:bg-slate-800 dark:text-slate-400">
  <div class="whitespace-break-spaces">Hey everyone!

It's almost 2022       and we still don't know if there       is aliens living among us, or do we? Maybe the person writing this is an alien.

You will never know.</div>
</div>
```

```html
<div class="w-3/4 ...">
  <div class="**whitespace-break-spaces** ...">Hey everyone!

It's almost 2022       and we still don't know if there       is aliens living among us, or do we? Maybe the person writing this is an alien.

You will never know.</div>
</div>
```
